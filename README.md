# Создание приложения Random Task Generator (Генератор случайных задач)
Это простое приложение с графическим интерфейсом (GUI), предназначенное для генерации случайных задач. Программа поможет вам разнообразить будни, предложив случайное занятие из предопределенного списка (например, «Прочитать статью» или «Сделать зарядку»). Приложение позволяет сохранять историю сгенерированных задач в файл формата JSON для последующего просмотра, а также фильтровать задачи по категориям: учеба, спорт или работа. Встроенная проверка ввода предотвращает добавление пустых задач.
Дайте пошаговую инструкцию для пользователя сплошным текстом. Например: чтобы запустить программу, откройте терминал в папке проекта и выполните команду python main.py. В главном окне нажмите кнопку «Сгенерировать задачу», чтобы получить случайный совет. Чтобы добавить собственную задачу, введите её текст в поле ввода и нажмите «Добавить». Вы можете отфильтровать отображаемые задачи по типу с помощью соответствующего меню. История ваших действий сохраняется в файл и автоматически загружается при следующем запуске приложения.
